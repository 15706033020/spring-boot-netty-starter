Netty-Starter

Netty快速开发的高性能IO整合框架

快速开始 →

快速开发

基于 Spring-boot 的 AutoConfiguration 功能,如果你不懂Netty,也无需担心,只需要简单的配置即可生成一个高性能的Socket服务器.

灵活定制

使用 Annotation , Lambda , Interceptor , Handler 等多种组合控制 IO Event .内置多种协议类型,通过配置即可使用.

性能高效

本地维护了一套 Session 管理,不用在为 Netty 的 Channel 管理而忧心,灵活配置连接的线程数量,通过配置参数即可调优.
