# 介绍

spring-boot-netty-starter是一款spring-boot和netty的整合框架，对于不太了解netty的同学，能够实现快速开发基于netty的socket服务器.其中实现了大量的Annotation和Process Interceptor,能够无缝对接spring-boot进行快速开发,接下来就让我们开始吧!

# 能做什么(What doing?)

对于新入java编程行业或者已经在这个行业拼搭的人员来说,IO属于基础理论,对于大部分人来说Netty这个框架其实并不陌生,但是原生的Netty框架于SpringBoot之间并无交集,而这个框架就是为了解决与SpringBoot的无缝整合.
